<TPS Gaige>

by Kazy

This mod changes Gaige's Skill Trees to be designed similarly to the Skill Trees seen in The Presequel.

This mod also changes Smaller, Lighter, Faster into a 3-Pointer, and increases Reload Speed to 9%. Robot Rampage has moved to Tier 1 to allow better DT usage.

This mod can be run after the UCP to gain all UCP skill boosts and the skill tree format.
