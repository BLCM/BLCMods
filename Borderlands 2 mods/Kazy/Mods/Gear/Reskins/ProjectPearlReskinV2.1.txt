<Project: RE-Pearl>

Project Pearl Reskin

//by Kazy

Changes the skins on the Pearlescent weapons

<Changelog [Current Version: V 2.1]>

<V 2.1>

Added an Optional Butcher Skin (On by Default. Resembles Old Hyperion)

Added an Optional Godfinger Skin (On by Default. Just changes the barrel to Jakobs but gains the Bekah's skin from 2.0)

Added an Optional Avenger Skin (On by Default. Now has a baby blue and dark gray color scheme and teal emissive effect)

Added an Optional Storm Skin (On by Default. Bears similarities to Aquamarine Snipers, but reversed stripes and bears an eTech barrel)

Added an Optional Sawbar Skin (On by Default. A rusted-like appearance with random accessories thrown on that do not grant bonuses.)

Added an Optional Bearcat Skin (On by Default. Follows the "Dahl" theme a little better for how skins are designed.)

Pondered Wanderlust Skin but decided forget it for now.

</V 2.1>

<V 2.0>

Added the Carnage

Added the Godfinger

Added the Bekah

Added an Optional Unforgiven Texture (On by Default. Slight Alteration)

Added Optional Red Text Change for Carnage: Holy crap! It shoots rockets!

Added Optional Name Text Change for Godfinger: Goldfinger

Added Optional Red Text Change for Godfinger: Such a Cold Finger.

</V 2.0>

Redesigned the Tunguska

Added the Sawbar

Added the Storm

</V 1.1>
