#<Better Splatguns>

Alters Splatguns to have bonus projectiles, bonus velocity, bonus fire rate, no self-damage, and no gravity. This mod eliminates the spawn chance of Tech 2 from all Splatguns due to it being useless. Please note this does not despawn existing Tech 2. Additionally, all Splatguns allow you to pick whether Splatguns do Reaper, Rocket, or Grenade splash. Please choose carefully, as these are not inter-compatible.

This mod is incompatible with Aaron0000's Splitters. This mod contains compatibility code for Bayonets On Slot 2 and Jakobs/Torgue Etech compatibility code is included in the Generic barrel section. Enable this after JTET if you use it.

Coded by Orudeon with major assistance from the_Nocturni. It is highly recommended you use LightChaosman's Filter Tool to install this.
