#<Revolvers>

    #<Description>

        By Aaron0000

        Features:

        Changes the Jakobs, Maliwan, and Torgue pistols to behave like the Revolvers of Borderlands 1.

        High damage.

        High accuracy.

        High critical hit damage (+50% on Torgue, +100% on Maliwan, +150% on Jakobs.)

        Low fire rate.

        High recoil.

        Consumes 3 ammo per shot (Maliwan consumes 4 per shot.)

        Averages six shots a clip (five for Maliwan).

        Double accessory is turned into a Masher accessory that only "mashes" the Revolvers, is a normal Double accessory otherwise.

        Accuracy accessory is turned into an accessory that behaves like Mag5 from Borderlands 1 (High damage, two-shots before reloading. Slower reloads, tighter crosshairs). Behaves like a normal accuracy accessory for non-Revolvers.

        NOTES;

        Couldn't get Tw0 Fang and such to have slower shots with Revolvers, so I flat out disabled double-shots on all Jakobs/Maliwan/Torgue Pistols (though I preserved the Devastator).

        Certain special guns have been given adjustments to better fit the changes caused by revolving the Jakobs/Maliwan/Torgue pistols. Disable if needed (such as for mods that go over those items). The items in question are;

        All Unique/Legendary/Pearlescent Jakobs pistols except for the Law (Maggie was adjusted so that the projectile count as a masher is x24 instead of x35.)

        Maliwan E-Tech pistols (just the ammo cost), Infection, and (Unofficial Community Patch) Wanderlust (UCP Wanderlust was adjusted so that the projectile count as a masher is x14 instead of x18.)

        All Unique/Legendary/Seraph Torgue pistols (Unkempt Harold was adjusted so that the projectile count as a masher is x5 instead of x6.)

        FIXES FOR OTHER MODS;

        If you want to use the standalone version of mashers (that "mashes" all pistols, instead of just Jakobs/Maliwan/Torgue), just activate Mashers after this mod.

        If using my E-Tech Pistols Remastered mod, there is a fix for the standard Maliwan e-tech pistols (or Jakobs and Torgue, if you have the mod for it) and the Wanderlust at the bottom.

        If using my Defiler mod, there is a fix for it at the bottom.

        If using my Discipline mod, there is a fix for it at the bottom.

        If using my Smasher mod, there is a fix for it at the bottom (to reduce the ammo cost).

        If using my 88 Fragnum mod, there is a fix for it at the bottom (to reduce the ammo cost).

        If using the Salvation from The Campin' Dutchpack mod, there is a fix for its damage (to compensate for the lack of critting) at the bottom.

        If using the Painbringer from The Campin' Dutchpack mod, there is a fix for it at the bottom.

        If using the Triple from The Campin' Dutchpack mod, there is a fix for each version of it at the bottom.

    #</Description>
