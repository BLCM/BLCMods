#<Classic Sledge's Shotgun>

    #<Description>

        By Aaron0000

        Turns Sledge's Shotgun into a recreation of his weapon from Borderlands 1.

        Features;

        Increased base damage.

        -1 projectile

        +15% multiplicative critical damage

        +10000% Burst Fire

        +100% Melee Damage (But 150% with the melee attachment)

        Decreased reload speed

        2-shot magazine regardless of variants; consumes one ammo per shot

        Always spawns with a Bayonet (but does not erase non-bayonet variants).

        Knockback returns!

        Two options for skins. Red skin is on by default.

        NOTE:

        If using S&S Forever, there is a compatibility version at the bottom.

        If using the Quadtastic mod, there is a compatibility fix for it located at the bottom.

        If using both S&S Forever and Quadtastic, there is a combined fix at the bottom.
